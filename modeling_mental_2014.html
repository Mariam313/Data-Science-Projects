Mental Health in Survey 2014

In this notebook, I take the cleaned dataset and perform feature engineering.

To reduce the number of features, I'm replacing the following featutres: 'benefits', 'anonymity', 'wellness_program', 'seek_help', with "compnay support score".

Reducing the number of features improves analysis and model's performance, and reduce computational costs.

To determine if features correlate, I will be converting the features into integers (0 for no, and 1 for yes).

Lastly, will use feature_importance from Random Forest and Log_reg to determine which features contribute more to mental health.
